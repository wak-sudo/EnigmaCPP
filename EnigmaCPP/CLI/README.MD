# CLI

Command line interface for EnigmaCPP.

## How to build

#### First 
Put:
- `EnigmaCPP.h`
- `EnigmaRotor.h`
- `EnigmaSettings.h`

from library project.

Inside "include" folder.

#### Second

Put `LibEnigmaCPP.a` into `libs` folder.

#### Third

Use `make`
